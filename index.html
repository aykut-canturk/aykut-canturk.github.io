🔧 Kurulum gerekli

Giriş sisteminin çalışması için Supabase bilgilerinizi eklemeniz gerekiyor.

  1. supabase.com'da ücretsiz bir proje açın.
  2. Settings → API (veya Data API)'den Project URL ve anon/publishable anahtarını kopyalayın.
  3. Bu dosyadaki SUPABASE_URL ve SUPABASE_ANON_KEY satırlarına yapıştırın.
  4. SQL Editor'de tabloyu ve güvenlik kurallarını oluşturun (kod dosyanın içinde yorum olarak var).
